# Notification App

A simple Windows notification application that displays clickable message bars.

## Features

- **Clickable Message Bars**: Each message is displayed as a horizontal bar that responds to clicks
- **Hover Effects**: Visual feedback when hovering over messages
- **Simple UI**: Clean, minimal interface using tkinter

## Usage

```bash
python3 notification_app.py
```

## Requirements

- Python 3.6+
- tkinter (included with Python on Windows)

## Current Features

- Displays 3 sample messages
- Click any message bar to trigger an event (prints to console)
- Hover effects for visual feedback

## Future Enhancements

- Webhook integration to receive real-time notifications
- Notification persistence
- Message dismissal
- Timestamp display
- Customizable styles
